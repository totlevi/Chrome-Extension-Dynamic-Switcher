Tab Switcher Dynamic
Firefox-style MRU tab switcher
How to use
Open / cycle forward CtrlQ
Cycle backward CtrlShiftQ
Navigate cards ←→  or  Tab
Jump to tab Enter
Cancel Esc
Release to jump Ctrl keyup
Why not Ctrl+Tab?

Chrome reserves Ctrl+Tab at the browser level — no extension can intercept it. The default here is Ctrl+Q. You can change it at chrome://extensions/shortcuts.
